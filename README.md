# ThreeJSDemo
学习WebGl相关知识，使用ThreeJS创建3d可视化展示

项目使用VScode服务插件`Live Server`启动。

`three.js`[GitHub地址](https://github.com/mrdoob/three.js/tree/master),所有使用到的库都是从上面cp下来放到本地的。你也可以选择CDN


操作说明：

1. 鼠标右键拖动移动位置。
2. 鼠标左键进行旋转。
3. 鼠标滚轮进行缩放。
